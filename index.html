What is CSS?

CSS (Cascading Style Sheets) is the code you use to style your webpage. CSS basics takes you through what you need to get started. We'll answer questions like: How do I make my text black or red? How do I make my content show up in such-and-such a place on the screen? How do I decorate my webpage with background images and colors?

Like HTML, CSS is not really a programming language. It is not a markup language either â€” it is a style sheet language. This means that it lets you apply styles selectively to elements in HTML documents. For example, to select all the paragraph elements on an HTML page and turn the text within them red, you'd write this CSS: 

          p {
            color: red;
          }
